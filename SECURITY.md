# Security Policy

## Supported Versions

Below are the versions of the script currently receiving security updates and active support.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.0   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

I take the security of this judicial system seriously. If you find a vulnerability or a potential exploit that could harm a server's database or player experience, please follow the steps below:

1. **Do not open a public GitHub Issue.** Publicly disclosing an exploit puts all servers at risk.
2. **Contact me directly** via Discord: **mcdikmen** or e-mail me on **muratcandikmen98@gmail.com**.
3. Provide a detailed description of the vulnerability and, if possible, steps to reproduce it.

### What to expect
- **Response Time:** I will acknowledge your report within 48 hours.
- **Fix Timeline:** Depending on the severity, I aim to release a patch within 3-7 days.
- **Credit:** If the vulnerability is verified, I will credit you in the next release notes (unless you prefer to remain anonymous).

Thank you for helping keep the Qbox community safe!
